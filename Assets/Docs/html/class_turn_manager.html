TurnManager TurnManager.Instance
static

TurnManager is a singleton that manages the turns of the game. It keeps track of the current turn, whether it's the first or second player's turn, and whether enhanced summoning is available for either player. It also manages the number of summons per turn and the turn number. The map parameters are used to determine the rules for summoning and turns. it pretty much manages the game flow and all turn related stuff.
