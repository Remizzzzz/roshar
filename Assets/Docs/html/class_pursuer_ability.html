  • This method is called every frame to check for user input and handle the ability activation. If the left mouse button is clicked and the ability is active, it checks if the clicked tile is a valid target. If it is, it moves the piece to that tile and boosts its attack, then resets the ability.

◆ abilityTargets

List<Vector3Int> PursuerAbility.abilityTargets =new()
private
  • This class implements the Pursuer ability, which allows a piece to move to a tile and boost its attack. It inherits from the Ability class and overrides the necessary methods to implement the specific logic for moving and boosting attack. The ability can only be activated during the player's turn and targets tiles within a range of 1 tile.
