TextMeshPro ButtonForInterruption.buttonText

This class is used to create a button for interruptions windows It is a simple button that can be clicked to perform an action. The button has a text that can be set, and a color that can be set. The button is scaled to fit the window. The button is a child of the interruption window. The button is used to perform an action (that can be set too) when clicked.
