◆ alliesTeleportable

int HeavenlyAbility.alliesTeleportable = 2
  • This class implements the Heavenly ability, which allows a piece to teleport allies to a tile. It inherits from the Ability class and overrides the necessary methods to implement the specific logic for teleporting allies. The ability can only be activated during the player's turn and targets allies within a range of 1 tile.
