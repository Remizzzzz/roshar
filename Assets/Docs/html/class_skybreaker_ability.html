Cost of the ability, to be defined in derived classes.

  • This method is called to activate the Skybreaker ability. It checks if the ability can be activated, sets the targets, and updates the piece state. If the ability is successfully activated, it sets the isAbilityActive flag to true.

◆ calculateDamage()

int SkybreakerAbility.calculateDamage ( )
  • This class implements the Skybreaker ability, which allows a piece to deal damage to an enemy piece. It inherits from the Ability class and overrides the necessary methods to implement the specific logic for dealing damage. The ability can only be activated during the player's turn and targets enemies within a range of 1 tile.
