This class is used to create a base tile that can be used in the game. It is a custom tile that can be used to create different types of tiles. It is used to create the base tile for the game. So it's mainly an editor script. It parameters each tile to change it's color based on the tile state. It also allows to shrink the tile to fit the grid better.
