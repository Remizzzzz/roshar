◆ Update()

override void StonewardAbility.Update ( )
protectedvirtual
  • This method is called every frame to check for user input and handle the ability activation. If the left mouse button is clicked and the ability is active, it checks if the clicked tile is a valid summoning tile. If it is, it summons a minion at the selected tile position and resets the ability.

◆ minionPrefab

GameObject StonewardAbility.minionPrefab
  • This class implements the Stoneward ability, which allows a piece to summon a minion on a tile. It inherits from the Ability class and overrides the necessary methods to implement the specific logic for summoning a minion. The ability can only be activated during the player's turn and targets tiles within a range of 1 tile.
