Member Function Documentation

◆ MovementPhase()

bool PhaseManager.MovementPhase ( )

The three methods : MovementPhase, SummoningPhase, and CombatPhase check if the current game phase is movement, summoning, or combat respectively. They return true if the current phase matches the specified phase.

◆ nextPhase()

void PhaseManager.nextPhase ( )

This method advances the game to the next phase. It checks the current phase and transitions to the next one accordingly. it also updates the AnimatePhaseChange flag to true, so the interface will animate the phase change.
