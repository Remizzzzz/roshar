At the instanciation of the \doxylink{class_window_interruption}{Window\+Interruption}, we set the text of the window and the buttons. The text is set to ask the player if they want to activate the ability of the caller. The buttons are set to accept or cancel the activation of the ability. The window is set to be in front of all other objects in the scene. The sorting order of the window is set to 50, so it is always in front of other objects. The sorting layer of the window is set to the same as the parent Sprite\+Renderer.

\doxylink{class_window_interruption}{Window\+Interruption} is a class that manages the interruption windows in the game. It creates a window with a text and two buttons (Yes and No (for now, but it is can be changed)) when an interruption occurs. The window is used to ask the player if they want to activate a specific ability. The buttons are used to accept or cancel the activation of the ability. The window is destroyed when the interruption ends.
