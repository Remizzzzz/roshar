\begin{DoxyItemize}
\item \doxylink{class_piece_state_manager}{Piece\+State\+Manager} is a singleton that manages the state of all pieces in the game. It keeps track of the state of each piece, whether it is a Fluctuomancer or a Fused piece. The states are stored in two dictionaries, one for Fluctuomancers and one for Fused pieces. The states can be updated, retrieved, and removed.
