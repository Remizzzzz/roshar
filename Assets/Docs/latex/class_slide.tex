void Slide.\+Start\+Animation (\begin{DoxyParamCaption}\item[{Transform}]{target}{}\end{DoxyParamCaption})}



Initial position of the object. 

Convert the \textquotesingle{}degree\textquotesingle{} value (in degrees) into a direction vector. Unity\textquotesingle{}s trigonometric functions use radians, so we convert degrees to radians. Cosine gives the X (horizontal) component of the direction. Sine gives the Y (vertical) component of the direction. This creates a 2D vector pointing in the specified angle on the XY plane. We normalize the vector to ensure its length is always 1, so that the slide speed remains consistent regardless of the angle.
