\doxylink{class_win_condition}{Win\+Condition} is a singleton that manages the win condition of the game. It keeps track of the number of Fluctuomancers and Fused pieces on the map. If there are no Fluctuomancers left, the Fused player wins, and vice versa. The win condition is checked every frame.
