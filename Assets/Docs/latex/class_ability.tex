\doxysubsection{Detailed Description}

\begin{DoxyItemize}
\item This class is the base class for all abilities in the game. It manages the stormlight and voidlight resources, ability activation, and restoration logic. It should be inherited by specific ability classes that implement the actual ability logic. 
\end{DoxyItemize}
